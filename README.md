## 启动

- 商城接口（后端）
- 商城页面（前端）

```sh
npm install --registry=https://registry.npmmirror.com
npm run serve
```



- 商城管理后台系统（前端）
```sh
npm install --registry=https://registry.npmmirror.com
npm run dev
```
